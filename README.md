# Portfolio
My personal project portfolio
Questo progetto mostra un cruscotto Power BI per l’analisi e il monitoraggio delle segnalazioni IT (ticket, service request e technical support).
I dati sono stati generati artificialmente simulando tre aree operative (Infrastructure & Hardware, Applications & Software, Security).
L’obiettivo è dimostrare capacità di:

Data cleaning e modellazione DAX

Creazione di un modello relazionale

Produzione di dashboard interattive

Storytelling e analisi operativa
Il file PBIX contiene misure, colonne calcolate e un modello dati strutturato secondo best practice.
